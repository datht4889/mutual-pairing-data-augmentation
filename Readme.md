# Mutual-pairing data augmentation for FCRE

## FCRE
### Change directory
```bash
cd FCRE
```

### Install requirements
```bash
pip install -r requirements.txt
```

### Run CPL (with bert and LLM)
```bash
cd CPL
bash bash.sh
bash bash_llm.sh
```

### Run SCKD
```bash
cd SCKD
bash bash.sh
```
### Run SCKD MI
```bash
cd CPL-mmi
bash bash.sh
```
### Run CPL MI
```bash
cd SCKD
bash bash.sh
```
